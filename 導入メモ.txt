→手順書作成

XAMPPはPHPのプログラミングに必要なソフトウェアをまとめてインストールし、各ソフトウェアの起動・停止を管理することのできるソフトウェアです。

導入はおおまかに以下の手順で進めます。

１．	インストーラのダウンロード
２．	インストールの実行
３．	XAMPPの起動を確認
４．	初期設定ファイルの編集

なお、インストールするパソコンのOSはWindowsの64bit版を想定しています。

XAMPPのダウンロード
まずXAMPPの公式サイトにアクセスし、公開されているインストーラを手に入れます。

XAMPP公式サイトへアクセス
『XAMPP』でWEB検索を行うか、アドレスバーに https://www.apachefriends.org/jp/index.html を入力し、トップページにアクセスします。

ダウンロード
「Windows 向け XAMPP」と表示されているボタンをクリックし、保存を実行します。
ダウンロード中はブラウザを閉じず、ダウンロード完了まで待機してください。
※保存の細かい手順はブラウザによって異なります。

ダウンロードファイルの確認
ダウンロードしたフォルダをエクスプローラで表示します。
デスクトップに保存した場合はデスクトップを表示します。
xampp-windows-x64-[バージョン]-installer.exe というファイルがあればダウンロードは成功しています。
※[バージョン]の部分はダウンロードのタイミングによって変わります。
※Google Chromeの場合、[Ctrl]+[J]でダウンロード履歴を確認し、ダウンロードフォルダを開くことができます。

XAMPPのインストール
ダウンロードしたexeファイルを実行してXAMPPをインストールします。

インストーラ実行
xampp-windows-x64-[バージョン]-installer.exe をダブルクリックして実行します。

インストールを進める
いくつかの設問のあとでインストールが実行されます。

①質問メッセージ
ウイルス対策ソフトウェアが起動している場合に表示されます。
「インストールに時間がかかる場合があるが、続行するか」という内容です。
→Yes

②警告メッセージ
インストール場所についての注意です。インストール場所をC:\Program Filesにしない場合は無視できます。
→OK

③Setupへようこそ
→Next

④インストールするソフトウェアの選択
→すべてにチェック Next

⑤インストール先フォルダの指定
デフォルトでC:\xamppが入力されています。
→C:\xamppのまま Next

⑥言語の選択
英語かドイツ語を選択できます。日本語はありません。
→左のアメリカ国旗を選択したまま Save

⑦Bitnamiの紹介
Bitnami for XAMPPというツールのWEBサイトを表示するかどうかのチェックボックスです。今回は不要です。
→チェックを外す Next

⑧Setupの準備完了
→Next

⑨Setup中
→（操作は不要です）

⑩Setup完了
すぐにXAMPPを起動するかどうかのチェックボックスです。
→チェック　Finish

インストーラが終了し、XAMPPコントロールパネルが立ち上がります。

ローカルサーバーの起動確認

ローカルサーバーの起動
 XAMPPコントロールパネルに
 Apach Start
 MySQL Start
ローカルサーバー起動の確認
 Apache Admin > Welcome to XAMPP
 MySQL Admin > phpMyAdmin
 画像のようにページが表示されれば起動は成功しています。(バージョンによっては見た目が異なる場合があります)
phpMyAdmin日本語化
 phpMyAdmin > 外観の設定 > 言語 > 日本語

php.iniの書き換え

php.iniの場所とバックアップ
 XAMPPコントロールパネル > Explorer
 php.iniを変更する前にファイルをコピーし、ファイル名を変更して保存しておきます。
php.ini書き換え
※設定
<ディレクティブ名> = <値> で設定
※コメントアウトについて
行頭の[;]はその行のコメントアウトを表している。コメントアウトされている行は設定に反映されません。
値を変更するときは、もともとあった行をコメントアウトして、その下に設定したいキーと値を書くとデフォルトがなんだったかすぐ確認できます。そうしなくても、バックアップしたphp.iniを見ることで確認できます。

以下のように変更してください。
※大体のエディターはctrl+Fで検索できる

○文字コード
文字化けが起こらないように
default_charset = UTF-8

○タイムゾーン
日本時間をデフォルトに設定
バージョンや導入環境によってはdate.timezoneが二か所
date.timezone=Asia/Tokyo

○マルチバイト文字の言語
全角文字を適切に扱えるように
mbstring.language = Japanese

○最大アップロードサイズ(40M)
upload_max_filesize=4M

○最大POSTサイズ(40M)
post_max_size=8M

○最大メモリサイズ(512M)
memory_limit=128M

初期状態でこれらの設定値を上回っていた場合についてはどうするか
upload_max_filesize=40M や memory_limit=512M など

※最大メモリサイズ(memory_limit)を超えるファイルは扱えないため、upload_max_filesizeはmemory_limitよりより大きな値にはなりません。
同じような理由で、これら３つの値は
upload_max_filesize ≦ post_max_size ≦ memory_limit
となります。

値はあとからでも変更できるため、必要に応じて設定しましょう。

phpinfoの確認
 Apacheを再起動
 Welcome to XAMPP > PHPInfo
 下の方までスクロール
 default_charsetやpost_max_sizeなど確認できる
 設定した値になっていたら、変更は完了しています。
